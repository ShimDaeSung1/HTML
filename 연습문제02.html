6. css 코드를 작성하는 방식 3가지를 설명하시오.
  • External Style Sheet : css라는 확장자를 가진 스타일 시트 파일을 만들고 이 파일을 HTML 문서에 연결하여 사용하는 방법
  • Internal Style Sheet : HTML문서내에서 head와 /head사이에 스타일을 정의하는 방법
  • Inline Styles : 위의 방법들에 비해서 적용범위가 더욱 좁아진 형태. 스타일을 적용하고 싶은 HTML태그안에서 정의하는 방법
